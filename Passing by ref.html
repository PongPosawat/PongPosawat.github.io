Comparison between PHP and Java/C/Python

1. Passing Arguments by Reference

PHP: In PHP, you can pass arguments by reference using the & symbol, which allows the function to modify the original variable.

function addFive(&$num) {
    $num += 5;
}

$original = 10;
addFive($original);
echo $original;  // Output: 15

Java: Java uses reference passing for objects, but primitive types are passed by value.

public class Example {
    public static void addFive(int num) {
        num += 5;
    }

    public static void main(String[] args) {
        int original = 10;
        addFive(original);
        System.out.println(original);  // Output: 10
    }
}

C: C uses pointers for reference passing.

void addFive(int* num) {
    *num += 5;
}

int main() {
    int original = 10;
    addFive(&original);
    printf("%d", original);  // Output: 15
    return 0;
}

Python: Python passes objects by reference for all types.

def add_five(num):
    return num + 5

original = 10
print(add_five(original))  // Output: 15

2. Variable Number of Arguments

PHP: You can pass a variable number of arguments using ....

function sum(...$numbers) {
    return array_sum($numbers);
}

echo sum(1, 2, 3, 4);  // Output: 10

Java: Similar to PHP, Java uses ... (varargs).

public static int sum(int... numbers) {
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return sum;
}

public static void main(String[] args) {
    System.out.println(sum(1, 2, 3, 4));  // Output: 10
}

C: C uses va_list for variable arguments.

#include <stdarg.h>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    int result = 0;
    for (int i = 0; i < count; i++) {
        result += va_arg(args, int);
    }
    va_end(args);
    return result;
}

int main() {
    printf("%d", sum(4, 1, 2, 3, 4));  // Output: 10
    return 0;
}

Python: Python uses *args for variable arguments.

def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  // Output: 10

3. PHP is a Loosely Typed Language

PHP: In PHP, you do not need to declare the type of variables, and they can change types dynamically.

$variable = 5;        // Integer
$variable = "Hello";  // String
echo $variable;       // Output: Hello

Java: Java is a strongly typed language, so you must declare variable types.

int num = 5;
num = "Hello";  // Error: incompatible types

C: Similar to Java, C is a strongly typed language.

int num = 5;
num = "Hello";  // Error: incompatible types

Python: Python is dynamically typed but strongly typed.

num = 5
num = "Hello"
print(num)  // Output: Hello

4. PHP Return Type Declarations

PHP: Since PHP 7.0, you can declare return types in function definitions.

function addNumbers(int $a, int $b): int {
    return $a + $b;
}

echo addNumbers(2, 3);  // Output: 5

Java: Java requires return types in all methods.

public static int addNumbers(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    System.out.println(addNumbers(2, 3));  // Output: 5
}

C: C requires return types in all functions.

int addNumbers(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", addNumbers(2, 3));  // Output: 5
    return 0;
}

Python: Python supports return type hints, but they are not enforced.

def add_numbers(a: int, b: int) -> int:
    return a + b

print(add_numbers(2, 3))  // Output: 5
